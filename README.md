About This Project
This project is based on the concept of digital literacy and which was part of my first year course. The main aim was to understand how to use digital tools in a proper and efficient way. Instead of learning theory this project gave me a chance to actually use different platforms and explore how they work in real life.

As a Student Digital Ambassador, I worked on different tasks like creating an infographic, creating my online profiles, practicing and exploring coding platforms, writing professional emails, and learning about cybercrime. 
Each task helped me understand something new about the digital world.


What This Repository Contains

report/ – My final project report explaining all the tasks  
task-1-presentation/ – Infographic made using canva  
task-2-portfolio/ – Screenshots of my github, linkedin, and stack overflow profiles  
task-3-platforms/ – Coding platform work and Google Form screenshots  
task-4-email-etiquette/ – Professional email drafts and social media guidelines  
task-5-cybercrime/– Case study and prevention tips related to cybercrime  



 Tools which i used

During this project, I used a mix of platforms to complete different tasks:
 Github for managing my work and repository  
 Linkedin to create a professional profile  
 Stack overflow to understand coding-related doubts  
 Hackerrank to practice basic coding  
 Canva to design the infographic  
 Google Forms to create a quiz  

 What I Learned

This project helped me in many ways. I learned how to build a digital profile, how to communicate professionally, and how to use online platforms more effectively. I also understood how important it is to stay safe online and be aware of cyber threats.
